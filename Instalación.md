# Instalación

# Debian / Ubuntu

Una vez configurados los repositorios correctamente y actualizado el sistema con apt update,
solo hay que escribir:

apt install wget

Para verificar la instalación:

dpkg -l | grep wget

Para validar la versión:

wget –-version
